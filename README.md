## Como instalar/executar

    Clone o repositório.

    > cd backend/nutripet

    > mvn clean package

    > docker-compose up --build

    Acesse a aplicação em um navegador na URL: http://localhost:3000/



## Tecnologias Utilizadas

- Maven
- SpringBoot
- MongoDB
- React.JS
- Docker-compose
